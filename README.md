# 🌿 Klasifikasi Rimpang Jamu Madura

Aplikasi ini dibuat untuk mengklasifikasikan lima jenis rimpang jamu Madura menggunakan metode **Transfer Learning VGG-19 sebagai Feature Extractor** dan **Random Forest Classifier**.

## Jenis Rimpang yang Didukung

- Jahe
- Kunyit
- Kencur
- Lengkuas
- Temulawak

## Cara Menggunakan

1. Buka halaman **Klasifikasi**.
2. Upload gambar rimpang (JPG, JPEG, PNG, atau HEIC).
3. Tunggu proses prediksi selesai.
4. Hasil klasifikasi akan ditampilkan pada layar.

## Teknologi

- Streamlit
- TensorFlow
- VGG-19
- Random Forest
- Scikit-learn
